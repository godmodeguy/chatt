# Chatt
> Simple tcp chat in go


### Commands
- `/name <username>` - set username, *anonymous[host:ip]* by default
- `/rooms` - list available room
- `/join <name> [password]` - join room
- `/quit` - exit from room
- `/newroom <name> [password] [--hidden]` - crete new room
